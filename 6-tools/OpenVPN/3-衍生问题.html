# 阿里云同VPC、同路由、多交换机

在阿里云同VPC、同路由、多交换机环境下。OpenVPN Server 搭建在 172.19.89.135 所属 172.19.80.0/20 交换机，访问不了其他另外两个网段的问题

首先排查一下当前路由表信息是否包含目标交换机的网段路由，使用命令 route -n
